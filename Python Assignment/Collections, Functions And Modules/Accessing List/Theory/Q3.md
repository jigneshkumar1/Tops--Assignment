Slicing allows you to access a subset of the list by specifying a range of indices.

 The syntax for slicing is:
                             list[start:end:step]


# Creating a list
lang = ['python', 'java', 'php', '.net']

# Slicing a list
print(lang[1:3])  # ['java','php']
